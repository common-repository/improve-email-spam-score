This plugin adds a email return-path and envelope-from address to the wp_mail () function. Which reduces the spam score sent from your website form.

== Description ==

This plugin adds a email return-path and envelope-from address to the wp_mail () function. Which reduces the spam score sent from your website form.

== Installation ==

Either using the dashboard 'Add Plugin' feature to find, install and activate the plugin,  or
1. Download and unzip the plugin
2. Upload the entire contents of improve-email-spam-score to the /wp-contents/plugins directory.
3. Activate the plugin through the Plugins menu
